# Harnessing-Dual-Role-Context

Our framework first processes hybrid text _S_ through a dynamic sliding-window mechanism to capture local context. Each analysis window represents a contiguous segment of length _len_ drawn from _S_.

The overall architecture is shown, which consists of three major modules: the semantic feature extraction module, the coherence feature extraction module and the confidence-enhanced detection module.
